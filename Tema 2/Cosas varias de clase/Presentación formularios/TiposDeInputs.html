Atributo Accept

Este atributo hace que este input solo acepte archivos de la extensión que queramos, por ejemplo .pdf

Atributo AutoFocus

Este atributo esta establecido para que sea lo primero que veas en el formulario

Atributo Checked

Este atributo hace que una casilla este marcada inicialmente

Atributo Disabled

Este campo no se puede rellenar

Atributo Min

Este campo tiene que tener un valor minimo de 10 (numerico)

Atributo Minlenght

Este campo tiene que tener una longitud minima de 10 (letras)

Atributo Max

Este campo tiene que tener un valor máximo de 20 (numerico)

Atributo Maxlenght

Este campo tiene que tener una longitud máxima de 20 (letras)

Atributo Multiple

Este campo puede soportar más de un fichero

Atributo Placeholder

Este campo muestra un texto que desaparece al escribir en el input

Atributo Readonly

Este campo muestra un texto que no puedes modificar

Atributo Required

Este campo es obligatorio para poder enviar el formulario

Atributo Size

El ancho de este input esta determinado por size

Atributo Value

El value es la información que manda un input al formulario
